🧪 INTEGRATION TESTING

1. Get OpusDNS sandbox API key:
   - Sign up at https://sandbox.opusdns.com
   - Create API key via dashboard or API

2. Create credentials file:
   $ mkdir -p ~/.secrets/certbot
   $ cat > ~/.secrets/certbot/opusdns.ini << 'CREDS'
   dns_opusdns_api_key = opk_your_sandbox_key_here
   dns_opusdns_api_endpoint = https://sandbox.opusdns.com
   CREDS
   $ chmod 600 ~/.secrets/certbot/opusdns.ini

3. Test with Let's Encrypt staging:
   $ certbot certonly \
       --authenticator dns-opusdns \
       --dns-opusdns-credentials ~/.secrets/certbot/opusdns.ini \
       --server https://acme-staging-v02.api.letsencrypt.org/directory \
       -d test.yourdomain.com
